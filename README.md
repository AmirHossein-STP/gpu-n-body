# GPU-Accelerated N-body Simulation

A GPU-accelerated N-body gravitational simulation using Python, Numba, and CUDA. Optimized particle interaction calculations through parallel execution on NVIDIA GPUs to improve simulation performance for large-scale dynamical systems.
